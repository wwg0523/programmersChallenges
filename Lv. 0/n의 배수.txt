n의 배수

<<< 입출력 예 >>>

num	n	result
98	2	1
34	3	0

<<< 입출력 예 설명 >>>

입출력 예 #1
98은 2의 배수이므로 1을 return합니다.

입출력 예 #2
32는 3의 배수가 아니므로 0을 return합니다.



<<< 내 풀이 >>>

class Solution {
    public int solution(int num, int n) {
        int answer = num % n == 0 ? 1 : 0;
        return answer;
    }
}
